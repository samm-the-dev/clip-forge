# Clip Forge

Video clip extraction pipeline: Keep notes → ffmpeg cuts → DaVinci Resolve → animated WebPs.

Use `/clip-forge <project-name>` to run the full guided pipeline.

## Completed Projects

- **One Piece S1** — 18 clips, TV show workflow
- **Frankenstein (2025)** — 5 clips, movie workflow, font: 1790 Royal Printing Caps
- **One Piece S2** — 18 clips, TV show workflow, RT Dyans font
